# 💖 Nene Immersive Chat (Alpha)

> 🧪 Experimental project — still in active development and may contain bugs.

An interactive AI character chat powered by LLM, featuring dynamic emotions, memory, and structured roleplay.

> A playful, immersive chat experience that feels alive — not just another chatbot.

---

## ✨ Features

* 🎭 **Character Persona System**
  Consistent personality with controlled behavior

* 🧠 **Emotion Engine**
  AI reacts based on user input (playful, shy, teasing, etc.)

* 💾 **Memory System**
  Relationship level evolves over time

* 🎬 **Structured Responses**
  Combines narration + action + dialogue for immersion

* 🔒 **Scenario Locking**
  Keeps the roleplay consistent (no context breaking)

---

## 🛠 Tech Stack

* Python
* Streamlit
* Ollama (Local LLM)
* Qwen 2.5 (7B Instruct)

---

## 🚀 Getting Started

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Run the app

```bash
streamlit run app.py
```

---

## ⚙️ Configuration

Make sure Ollama is running locally:

```bash
ollama run qcwind/qwen2.5-7B-instruct-Q4_K_M
```
